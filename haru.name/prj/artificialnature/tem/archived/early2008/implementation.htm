Art Installation & Research Project, Haru Ji & Graham Wakefield, 2007-2008

Abstract | Concepts | Prototypes | Implementation | Artists

Implementation

Artificial Nature is an ongoing research project, with a number of actual and potential realizations.

 

Software development

Early prototypes of the project have been implemented using the Max/MSP/Jitter environment, along with Lua scripting extensions (jit.gl.lua and lua~ for graphics and audio respectively).

The current iterations of the project however are entirely written in portable C/C++ with bindings for the Lua scripting language. It is a standalone executable application (portable to MacOSX, Windows or LInux) for continuous generative production of spatialized audio (for two to sixteen loudspeakers) and multiple displays. Graphics are rendered using OpenGL, and audio is synthesized using custom-written C++ signal processing and spatialization.

The use of the Lua programming language has been central to the rapid development of the project. Lua is not only a flexible language for prototyping, it is also a highly efficient virtual machine with dynamic typing and an incremental garbage collector. We make use of Lua tables and coroutines extensively to expressively model the interactions of cells, organisms, genes, chemicals, sounds and other dynamic elements in the installation.

 

Installation Description

The installation consists of multiple projections and spatialized audio providing inner and outer viewpoints upon the simulation as it progresses in real time. The simulation itself runs upon a Mac Pro embedded within the central table. The table surface is a Wacom Cintiq or touch screen offering direct interaction with the field environment. Additional sensors are placed around the screen.
